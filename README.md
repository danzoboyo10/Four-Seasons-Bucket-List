## Four-Seasons-Bucket-List

A fun and interactive bucket list application that allows you to create, read, update, and delete bucket list items!

## Technologies Used 
* HTML 5
* CSS 3
* JavaScript
* MongoDB
* Mongoose
* Node.js
* Express
